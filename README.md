## Available Scripts

In the project directory, you can run:
### `npm install`
Installs the necessary server and client dependecies required for the app to function.

This app uses following packages to implement the API server for the test data.

- [json-server](https://www.npmjs.com/package/json-server) Creates a REST server from the data stored in db.json
- [concurrently](https://www.npmjs.com/package/concurrently) starts application server along with react development.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
